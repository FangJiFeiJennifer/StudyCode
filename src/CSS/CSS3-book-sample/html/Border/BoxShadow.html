box-shadow属性向框添加一个或多个阴影，有2~6个参数可选，必须的参数是水平位移和纵向位移。 box-shadow: h-shadow v-shadow blur spread color inset;

值 描述
h-shadow 必需，水平阴影偏移量，允许负值。取正直阴影在元素的右边，反之取负值阴影在元素的左边。
v-shadow 必需，垂直阴影偏移量，允许负值。取正直阴影在元素的下边，反之取负值阴影在元素的上边。
blur 可选，模糊距离。其值只能是正值，如果为0，表示不具有模糊效果，值越大阴影边缘越模糊。
spread 可选，阴影的扩展半径，可以取负值。值越大阴影越大，反之取负值则整个阴影都缩小。
color 可选，阴影的颜色。
inset 可选，将外部（outset）阴影改成内部阴影。默认是outset.

利用阴影设置元素的边框不会影响页面的布局，不同于border会被计算，因此可以用box-shadow自有的模拟元素边框，但是必须注意其层级关系。

  1. 非零值的border-radius会以相同的作用影响阴影的外形。
  2. border-image不会影响对象阴影的任何外形。
  3. 外阴影会在对象背景之上，内阴影会在边框之下，背景之上。
  4. 总之：边框在内阴影之上，内阴影在背景图片之上，背景图片在背景色之上，背景色在外阴影之上。

多层盒阴影

  1. 制作多层阴影，不设置模糊半径，只设置扩展半径,并配合多个阴影颜色，可以制作多色边框效果，替代border-color制作的多色边框效果。
  2. box-shadow制作多色边框时需要注意，在计算宽度时需要减去前面的阴影的值，才是显示的颜色宽度。
  3. 制作多层box-shadow时要注意阴影的顺序，最先写的阴影将显示在最顶层。
