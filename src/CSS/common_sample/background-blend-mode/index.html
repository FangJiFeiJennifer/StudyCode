blend-mode 概 念

  1. 使用 background-blend-mode: lighten。 lighten : 变亮, 最终颜色是由每个颜色通道的最亮值组成的颜色.
    其实就是图片或者背景色重叠的时候，颜色更亮的显示出来，颜色暗的被覆盖掉。
    顾名思义，这个混合只是混合背景色，不会影响元素的前景内容。

    (1). 实现白色底黑色图案图片颜色赋色技术，图片格式无限制，JPG、PNG、GIF 都可以

    • 黑色是最暗的颜色，所以可以被任何颜色替换掉;
    • 白色最亮，所以不会有任何变化;
    • 反之，如果素材的底色是黑色，主色是白色。那就应该用变暗（darken）的混合模式.
