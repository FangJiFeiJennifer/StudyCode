1. 内联元素默认是不会换行的，除非元素的宽度超过了屏幕的宽度。
  2. CSS为内联元素设置的高和宽是不起作用的，除非为内联元素设置display:inline-block;
  3. 内联元素之间默认左右有间距，上下没有。
  4. 通过以下显示内容，我们可以看出内联元素与inline-block之间的区别。
