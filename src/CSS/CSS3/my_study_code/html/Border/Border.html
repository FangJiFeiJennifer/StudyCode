CSS3 border-image

  1. border-image: none | <image-source> [number | percentage] {1,4} [border-width] [stretch | repeat | round] {0,2}
  2. none是默认值，表示无边框图片。
  3. image-source是指图片路径。
  4. number是一个数值，用来设置边框或者边框背景图片的大小，单位是px，可以是1~4个值，表示四个方位（上，右，下，左）。
  5. percentage跟number作用相同，不同之处是percentage用的是百分比。
  6. stretch,repeat,round设置背景图片的平铺方式。stretch拉伸边框背景图片，为默认值；round是平铺；repeat是重复。
border-radius > border-width内半径大于零，出现内圆角。
border-radius < border-width内半径小于零，没有内圆角。
元素相邻边有不同宽度，圆角将会从宽的边平滑过渡到窄的一边，元素相邻转角是由大到小。

利用圆角画圆时，元素的高度和宽度相同，圆角的半径值为元素高或者宽的一半，或者直接设置为50%

用border-radius做椭圆需要设置水平和垂直方向的半径值。 水平椭圆元素的宽度是高度的两倍，而且border-radius的水平半径等于元素的宽度，垂直半径等于元素高度； 而垂直椭圆的参数刚好相反。
